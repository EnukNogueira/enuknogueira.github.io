#  Portfólio - Enuk 

Este é o meu hub central de projetos e competências técnicas, focado em **Back-End, Engenharia de Dados, Automação e Inteligência Artificial**. 

O objetivo deste site é servir como uma interface rápida e eficiente para visualizar meu progresso na transição da área industrial para o ecossistema de tecnologia.

##  Desenvolvimento Assistido por IA
Este site foi desenvolvido através de um processo de **Orquestração de IA Agêntica**. Em vez de utilizar templates prontos, utilizei agentes autônomos de inteligência artificial para:
- Estruturar o código HTML5 e CSS3 moderno.
- Gerar uma interface responsiva com estética Cyberpunk/Terminal.
- Otimizar a descrição técnica dos projetos baseando-se nos meus repositórios reais.

Minha função foi atuar como **Tech Lead**, definindo as diretrizes, corrigindo a lógica e garantindo que o resultado final refletisse minha identidade profissional e rigor técnico.

##  Stack Tecnológica do Site
- **Linguagens:** HTML5, CSS3.
- **Estilo:** Design minimalista com foco em legibilidade e performance.
- **Hospedagem:** GitHub Pages.
- **Conceito:** Interface de Comando Industrial (alusão à minha base em Automação Industrial).

##  Como navegar?
- **Sobre Mim:** Um resumo da minha trajetória de transição de carreira.
- **Stack Técnica:** As ferramentas que domino no dia a dia (Java, Python, SQL, Docker).
- **Projetos:** Links diretos para os meus repositórios no GitHub, onde a lógica de Back-End acontece de verdade.
